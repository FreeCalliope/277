
The objective of this paper is to implement ML methods to predict reaction yield from features using neural networks, XGB regression, random forest regression and to classify yield into categories (low, high, etc.) 

All data supporting this project is stored at this  Github repository, also linked in the beginning of this paper.
Exploration of the data can happen through any of the three .ipynb notebooks; results given in the appendices are pre-loaded into these notebooks. 
Local copies of Open Reaction Database Data are also uploaded to the repository to support code execution, in the files starting with ord_dataset. 
All dataset references are also included with hyperlinks in datasets.txt. 

To begin, visit any notebook beginning with "Methods On." Results are pre-loaded there, and instructions to operate the code are also included. 
Several imports will be needed, including Scikit, NumPY, Rdkit, Tensor Flow, Matplot, Pandas and Wget. All required libraries are installed and imported in each Methods file.
